Welcome to ChromeStrapp - the packaged Chrome app boilerplate.

The aim of this project is create awesome desktop apps, where Google Chrome is the only dependancy.

ChromeStrapp attempts to provide a set of widgets and styles that you can use to quickly produce good looking desktop applications.
